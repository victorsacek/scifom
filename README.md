# SCIFOM #

Surface processes, Climate, Isostasy, Flexure, Orogeny and Mantle dynamics code

### Steps to compile the code ###

* Clone the SCIFOM repository.

* Compile the code in the `scifom/src/` directory:

    ```
    make all
    ```
